C implementation of the Conway's game of life.

---

### Dependencies
- **libx11-dev**

---

### Get this game
```bash
  $ git clone https://github.com/Mastro-Gibbs/CGOL.git
  $ cd CGOL
  $ ./configure
  $ make
```
---

### How to
run **./cgol --help** for all launch/runtime options
